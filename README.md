# Aplikasi CRUD Mahasiswa (PHP OOP + PDO)

## Deskripsi
Aplikasi backend sederhana untuk mengelola data **Mahasiswa** (CRUD).  
Entitas: `mahasiswa` dengan atribut: `nama`, `nim`, `prodi`, `angkatan`, `foto`, `status`.

## Teknologi
- PHP 8.x
- MySQL / MariaDB
- PDO (PHP Data Objects)
